Functional styling in Flutter like Swift UI
## Features
* ✅ Text styling
* ✅ Widget styling
* ✅ and more
## Getting started

This package is in development mode.

## Usage

```dart
class HomePage extends StatelessWidget {
  const HomePage({super.key});

  @override
  Widget build(context) {
    return Scaffold(
      body: 'hello world'
          .text
          .color(Colors.amber)
          .fontFamily('family')
          .render,
    );
  }
}
```
